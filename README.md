# AI-Driven Workforce Income Classification System

An advanced Machine Learning prototype designed to predict and classify annual income brackets using a localized Indian workforce context.

## Tech Stack
* Core: Python 3.10+
* ML: Scikit-Learn (Random Forest Classifier)
* Data: Pandas & NumPy
* UI: Gradio (Web Dashboard)

## Key Features
* Localized UX: Custom mapping of census data to Indian educational and professional tiers (e.g., B.Tech, LPA).
* Ensemble Learning: Utilizes 100 Decision Trees to achieve ~85% prediction accuracy.
* Futuristic UI: Dark-mode dashboard interface optimized for accessibility.

## Results
Mathematical analysis identified Education Level and Age as the primary indicators of wealth distribution. The project bridges the gap between complex raw data and intuitive end-user tools.
